Folders for scoville repository:

/etc/scvrepo/ -- configfiles for scovillerepo. place config.json here
/usr/share/scvrepo/ -- datafiles for scovillerepo. place template.html and static/repo.png here

Database Generation:
    Run the repo_database.sql on an empty Firebird Database
    the default user for this firebird database is SCVREPO
    if you desire to use another username please change the settings
    in config.json and in repo_database.sql
